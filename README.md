# transaction.py
將自己的投資組合，透過籌碼面、技術面 & 基本面與大盤績效做比較，做出個股相對比較的對照表，包括股價走勢、累計報酬、超額績效、法人籌碼、融資融券借券、基本面分析與綜合建議表，以及設立各式條件與分數評比去篩選個股適合買進還是賣出。
# market index analyze.py
利用總體經濟指標與判斷條件來提醒進出場訊號，並利用總體趨勢判斷近3個月均值vs前三個月均值來判定股市的上升與下降。
# PTP all.py
建立PTP商品總集合，並以排成方式每月更新標的，同時結合寄信通知功能，提升作業效率。
# ELN.py
利用上手信建立結構債的通知流程與流程自動化，將提前解約、到期結算與轉換股票的通知，利用自動化的方式通知持有該商品的客戶，節省作業時間。
# ETF.py
依據市場ETF的成交值、成交量、日均量與均價表現，評估第一金證券ETF標的的新增與剔除。
